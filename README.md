更新说明：
# IPQMD 访客天气签名图组件
纯腾讯地图API实现，无高德依赖，轻量化PHP访客信息展示卡片，适配博客、个人主页侧边栏嵌入。

## 项目简介
基于腾讯位置服务一套密钥完成「IP属地解析 + IPv6兼容 + 实时天气查询」，内置多级缓存降低接口消耗，支持WebP/PNG/JPG素材、自动批量背景图、防盗链、自定义标题、DIY自定义文字参数，无需数据库，上传即可部署。

### 配额对比（2026个人开发者免费额度）
1. **旧方案（腾讯定位+高德天气）**
   - 腾讯IP定位：6000次/天
   - 高德天气预报：仅5000次/月（严重瓶颈）
   - 需两套密钥、两套域名白名单，易出现鉴权报错

2. **本项目纯腾讯方案（推荐）**
   - 腾讯IP定位：6000次/日
   - 腾讯实时天气：6000次/日（独立额度，互不占用）
   - 月度总可用约36万次，无月度限额短板，仅需1组Key

## 核心功能
1. **访客IP解析**
   - 自动识别CDN反向代理真实访客IP，同时兼容IPv4/IPv6地址
   - 独立IP缓存目录，单访客30分钟缓存，减少定位接口请求
2. **同城共享实时天气**
   - 依据城市adcode缓存天气数据，同一城市所有访客共用一份缓存
   - 展示：天气状况、实时温度、湿度、风向、风力、气象更新时间
   - 内置晴/雨/雪/雾/霾/多云/阴天对应天气图标
3. **设备识别展示**
   - 自动识别系统：Windows10/11、鸿蒙、iOS、Android
   - 识别主流浏览器：Chrome、Edge、Firefox、QQ浏览器、360、Safari
   - IPv6访问自动隐藏浏览器图标，避免画面拥挤文字重叠
4. **素材全面兼容WebP**
   - 背景图、天气图标、侧边小图标均支持 `webp/png/jpg/jpeg`
   - 背景图自动扫描 `img/bg*` 格式文件，新增图片无需修改代码
5. **自定义功能**
   - 配置文件自定义站点顶部欢迎标语
   - URL传参DIY自定义标题：`?diy=base64编码文字`
6. **安全与防盗**
   - 配置与代码分离，真实密钥存放`config.php`，模板`config.example.php`开源上传
   - 内置防盗链开关，可禁止外部直接图片外链盗刷API额度
   - CURL请求携带备案域名Referer，解决腾讯`key格式错误/110未授权`报错
7. **容错优化**
   - 字体、图片缺失自动兜底空白画布，不会500崩溃
   - CURL超时限制5秒，防止接口加载卡死页面
   - 自动递归创建缓存目录，自动权限0777适配虚拟主机

## 部署步骤
1. 下载全部源码，将`ipqmd`文件夹上传至网站根目录
2. 复制`config.example.php`，重命名为`config.php`
3. 前往腾讯位置服务后台申请WebService密钥，填入配置文件
   - 后台添加你的完整域名至白名单（带`https://`）
4. 按需调整防盗链开关、站点自定义标题
5. 在博客/网页嵌入图片标签直接调用

缓存说明
默认缓存时长：1800 秒（30 分钟）
IP 定位缓存：单访客 30 分钟，避免重复解析 IP
天气缓存：按城市共享 30 分钟，平衡天气实时性与接口消耗
腾讯气象数据源更新间隔约 10~15 分钟，30 分钟缓存兼顾新鲜度与额度
如需延长缓存，修改类内 private $cache_ttl = 1800; 数值（单位秒）
输出格式切换
程序默认输出 GIF 图片，全平台兼容（微信、QQ、老旧浏览器无裂图）；
如需更小体积 WebP 图片，打开index.php找到 suture 方法，取消 WebP 代码注释即可切换。
常见报错排查
status:301 key缺少
config.php 中$tencent_map_key未填写完整腾讯密钥，不要留空。
status:311 key格式错误
密钥复制存在空格、换行；或后台未创建「WebService」类型密钥。
110 域名未授权
腾讯后台添加网站完整域名白名单，config.php内$site_referer填写一致域名。
图片空白 / 文字不显示
检查font/msyh.ttf字体文件存在，服务器 GD、WebP 扩展正常开启。
