一款具有显示天气的IP签名档

- 源码内文件修改有详细解释
- 天气等有图标显示
- 具有风向湿度等提示

原作者：https://github.com/DeerShark/Liues-IPCard
grbj进行修改，修改说明：https://www.grbj.cn/998.html
本版基于grbj修改。

使用方法：
1. 将压缩包上传网站根目录解压缩
2. 将ipqmd文件夹内config.example.php重命名为config.php
3. 打开config.php填入相关内容
4. 在需要的位置插入html代码："<img src="https://www.example.com/ipqmd/index.php" style="width:100%;border:none;border-radius:5px;" alt="IP签名图">"
